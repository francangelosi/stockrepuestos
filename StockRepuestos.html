Configuración inicial — solo se hace una vez
Seguí estos pasos:
  1. Abrí console.firebase.google.com
  2. Menú izquierdo → Bases de datos → Realtime Database → Crear base de datos → Modo de prueba → Habilitar
  3. Menú izquierdo → Seguridad → Authentication → Comenzar → Correo/contraseña → Habilitar → Guardar
  4. Ícono ⚙ → General → abajo en "Tus apps" → Agregar app → </> Web → registrar → copiá el firebaseConfig
